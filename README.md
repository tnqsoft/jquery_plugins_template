# JQuery Plugins Template With Typescript

Write JQuery plugins with Typescript. 

## Setup
```
npm i -d
```

## Build
```
npm run build
```

## Prepare Demo
```
npm run preparedemo
```

## Release
```
npm run release
```

## TSLint
```
npm run tslint
```

## TS Watch
```
npm run tsc:w
```
